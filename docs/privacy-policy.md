# 隐私政策

**最后更新日期：2026年6月30日**

生效日期：2026年7月1日



欢迎使用 NovaAPI（以下简称“我们”或“本平台”）。我们深知数据隐私对于开发者与企业客户的极端重要性。作为一家提供全球大模型聚合路由以及特定垂类API与Agent服务的平台，我们致力于通过最高标准的安全措施保护您的信息。

本隐私保护政策旨在向您说明我们在提供产品或服务时如何收集、使用、存储、删除您的个人信息，并提示您所享有的个人信息权利。

**在您开始使用我们的网站和服务前，请您务必先仔细阅读和理解本隐私政策，确保您充分理解和同意之后再开始使用。**



### 1\. 我们收集什么信息？

为了提供安全、稳定的计费与服务，我们**仅收集维持服务运转的最低限度信息**：

- **账户与结算信息**

    - **基础凭证**：您的注册电子邮件地址及账户密码（哈希加密加密存储）**。**

    - **账单信息**：当您进行充值时，支付处理由安全的第三方名义商家（如 Stripe, Lemon Squeezy, Paddle 等）完成。**我们不直接收集或存储您的完整信用卡号等敏感金融数据。**

- **API 使用元数据 \(Metadata\)**

    - 为了实现精确的按量计费（Pay\-as\-you\-go），我们会记录每次 API 调用的**元数据**：请求时间、请求 IP 地址、调用的目标模型名称（如 `deepseek-v4-pro`）、以及输入/输出的 Token 消耗量。

- **系统技术信息**

    - 用于安全防御及系统优化的设备信息（浏览器类型、操作系统）及报错日志。

        

### 2\. 我们如何使用您的信息？

本平台收集您的个人信息，将用于以下用途。当我们要将收集到的信息用于本政策未载明的其他用途时，会事先征求您的同意和授权。

1. 向您提供、维护、改进产品或服务；

2. 向您发送关于系统安全、产品或服务变化等方面的重要通知；

3. 激活或验证产品或服务；

4. 在法律法规允许的前提下向您推广、介绍产品；您有权选择退出；

5. 为您提供个性化产品、服务或个性化用户体验；

6. 开展内部审计、数据分析和研究，以改善我们的产品和服务；您在自动化个人决策和用户画像方面的权利；

7. 保护我们产品、服务和用户的信息技术安全；

8. 遵从和执行适用的法律要求或相关的行业标准，如法律有关于实名认证的规定，所收集的包括银行卡号在内的信息都只用于实名认证用途，而不会用于其他目的；

9. 经您许可的其他用途。如您对数据处理用途有疑义，或希望我们停止其中一种或多种数据使用方式，请与客服团队联系。



### 3\. 我们如何共享信息？

我们**绝不出售、出租**您的任何个人数据。我们仅在以下必要场景下与受信任的第三方共享信息：

- **模型供应商：接收完成推理所需的用户内容及技术信息。**他们将根据其各自的隐私政策处理数据。

- **云与基础设施供应商**：提供服务器、数据库、对象存储、CDN、DDoS 防护、监控、日志、邮件和客户支持。它们受严格的数据处理协议（DPA）约束。

- **政府、监管和执法机构**：仅在有合法依据、具备适当权限且请求范围合理时披露；在法律允许时，我们会审查、缩小或质疑过度请求。



### 4\. 数据安全与存储

- **传输加密**：所有通过 NovaAPI 进出的流量均通过强标准的 TLS/SSL 端到端加密（HTTPS）。

- **密钥脱敏**：您生成的 `sk-` API Key 及上游渠道密钥在数据库中均采用行业最高标准的非对称加密存储。

- **没有绝对的安全**：尽管我们采用了顶级的商业防护措施，但互联网传输无法保证 100% 绝对安全。请您务必妥善保管自己的 API Key，避免在公开代码仓库中硬编码泄露。

    

### 5\. 数据保留期限

- **账户数据**：只要您的账户处于活跃状态，我们将一直保留您的注册信息。

- **日志数据**：为处理账单争议及异常调用分析，API 调用的元数据（仅限时间、模型、Token数）**最多保留 90 天**，随后将自动滚动删除或完全匿名化。

- **注销清除**：当您主动申请注销账户后，我们将在 30 日内从活动数据库中永久抹除您的个人标识信息。

    

### 6\. 您的权利

依据您所在地法律及我们所扮演的角色，您可能有权：

- 知情、访问并获取个人信息副本；

- 更正或补充不准确的信息；

- 删除个人信息或注销账户；

- 限制或反对特定处理；

- 撤回同意；

- 获取结构化、常用且机器可读的数据，或请求转移；

- 了解自动化决策的基本情况，并在法律规定时拒绝仅由自动化决策作出的重大决定；

- 退订营销信息；

- 就隐私处理提出投诉、申诉或向监管机构投诉。

如需行使上述权利，请通过本政策末尾的联系方式与我们联系。



### 7\. Cookie 与类似技术

我们可能使用：

- 登录与会话 Cookie，用于保持登录状态和安全退出；

- 安全 Cookie，用于防机器人、DDoS、欺诈和异常登录；

- 偏好 Cookie，用于记住语言和界面设置；

- 分析 Cookie，用于了解网站性能和使用情况，但仅在适用法律允许或您同意后启用。

我们不以出售个人信息为目的使用 Cookie，并支持用户管理非必要 Cookie。



### 8\. 跨境数据传输

您的信息可能被传输至您所在国家或地区以外的服务器进行处理。在此情况下，我们会采取适当措施确保您的信息得到与本隐私政策一致的保护。我们将在符合适用法律要求的前提下进行跨境数据传输。



### 9\. 政策的更新

我们可能因服务、供应商、技术或法律变化更新本政策。重大变更将通过电子邮件、控制台或网站提前合理通知；法律要求时会重新取得同意。页面顶部将显示最新更新和生效日期。继续使用 NovaAPI，即表明您接受更新后的隐私条款。
